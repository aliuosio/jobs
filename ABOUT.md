# About

**Job Forms Helper** is an AI-powered browser extension and API that automatically fills job application forms using your resume data. Built with a RAG (Retrieval-Augmented Generation) pipeline, it intelligently matches form fields with relevant resume content using Mistral AI embeddings and semantic search.

## Key Capabilities

- 🤖 **AI-Powered Form Filling** - Automatically detects and fills job application fields
- 📄 **Resume-Based Responses** - Generates contextual answers from your stored resume data
- 🔍 **Smart Field Detection** - 5-strategy label detection for accurate field matching
- 📊 **Job Tracking** - Manage and track job applications with real-time status sync
